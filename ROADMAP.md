# Roadmap

## V1: Free Local Builder Tool

- Prompt refinement before GitHub search.
- Human-readable top repo explanations.
- Save boards for local repo research.
- Markdown report export.
- Build Pack generation for Codex, Claude Code, Cursor, Lovable, and generic AI builders.
- BYOK support for GitHub, OpenAI, Groq, DeepSeek, and custom OpenAI-compatible providers.

## V1.1: Better Research Workflow

- More filters for current chat leads.
- Better duplicate detection across searches.
- Repo freshness and release-health signals.
- Saved research collections per idea.
- Screenshot/GIF demo assets for the public README.

## V2: Hosted/Advanced Version

- Optional cloud accounts and synced saved boards.
- Advanced internet research beyond GitHub.
- Private repo and organization search with user authorization.
- Team sharing and export links.
- Paid hosted tier with managed keys, stronger search, and persistent cloud history.

## Non-Goals For V1

- No write access to user repositories.
- No private repo access.
- No hosted user accounts.
- No paid tier in the open-source local version.
